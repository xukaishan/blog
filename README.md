# A personal blog based on vuepress

Blog theme based on vuepress-theme-reco-cli


## Use

```bash
git clone https://github.com/xukaishan/blog.git

cd blog

npm install 

npm run dev
```

if yarn, you can do this way:

```bash
git clone https://github.com/xukaishan/blog.git

cd blog

yarn install

yarn dev
```
